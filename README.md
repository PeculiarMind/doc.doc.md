# doc.doc.md

## Purpose
This project provides a set of bash scripts to orchestrate tools that extract metadata and information from files, with the goal of writing summaries in Markdown format.

## Project Status
Early-stage repository. The vision and documentation exist, and the bash tooling is expected to live in this repository as it evolves.

## Setup / Usage
No installation steps are documented yet. The intended usage looks like this:

```
./doc.doc.sh -d <directory_to_analyze> -m <report_template> -v
```

When the scripts are added, update this section with prerequisites, installation steps, and concrete examples.
